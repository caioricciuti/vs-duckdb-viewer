# Changelog

## 0.1.0 — 2026-04-05

Initial release.

- Right-click `.db`, `.duckdb`, or `.ddb` files to open with DuckDB Viewer
- Sidebar listing all tables with row count badges
- Paginated data grid (100 rows/page) with column types and row numbers
- SQL query editor with Cmd/Ctrl+Enter execution
- Column resizing via drag handles
- Copy results as CSV
- Refresh button to reconnect and reload tables
- Query execution time display
- Full VS Code theme support (dark/light)
